# 🚀 Immersed — Real-Time Focus Platform

> Stop guessing your productivity. Start **measuring it.**

Immersed is a real-time focus tracking platform that detects distractions, tracks consistency, and turns productivity into a **competitive, gamified experience**.

🌐 Live: https://getimmersed.in

---

## ⚡ What makes it different?

Most focus apps:
- Just run a timer ⏱️
- Assume you're focused

Immersed actually tracks:
- ❌ When you leave the tab
- ⏳ How long you're distracted
- 📊 Your real focus percentage

---

## 🧠 Core Features

- 🔴 **Real-time focus tracking**
- ⚠️ **Distraction detection (tab switching / inactivity)**
- 📊 **Focus score (%) based on behavior**
- 🏆 **Global leaderboard**
- 👥 **Live focus rooms (compete with friends)**
- 🎯 **Gamified productivity system**
